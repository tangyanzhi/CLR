# 怎么学？
持续学习才是技术的根本，有几个学习方法，跟大家分享下
1.多看微软官方教程，最权威的。
2.多上国外的网站，比如油管，谷歌技术论坛，Linux开源社区，在那里你能见到最前沿的，最高水平的编程人员。也能学到很多东西。
3.自己多练习，多做多写，古话云：光说不练假把式。

最后，祝大家技术更上一层楼。
